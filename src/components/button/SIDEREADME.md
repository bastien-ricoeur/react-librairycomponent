##### Required props

| Name       | Type   | Description                 |
| ---------- | ------ | --------------------------- |
| `children` | `node` | e.g: Add a label or an icon |

##### Optional props

| Name         | Type       | Default    | Description               |
| ------------ | ---------- | ---------- | ------------------------- |
| `onClick`    | `function` | `() => {}` |                           |
| `disabled`   | `boolean`  | `false`    |                           |
| `type`       | `string`   | `submit`   |                           |
| `color`      | `string`   | `button`   | `Either an Hex or a name` |
| `dataId`     | `string`   | `button`   | `specify the action`      |
