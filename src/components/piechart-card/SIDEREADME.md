##### Required props

| Name          | Type     | Description                                               |
| ------------- | -------- | --------------------------------------------------------- |
| `label`       | `string` | `The card label`                                          |
| `data`        | `array`  | `[{label: 'Data 1',data: 90},{label: 'Data 2',data: 10}]` |

##### Optional props

| Name   | Type                       | Default  | Description            |
| ------ | -------------------------- | -------- | ---------------------- |
| `size` | `enum(small,medium,large)` | `medium` | `The size of the card` |
